# Repo Workflow

- Small task: implement directly.
- Medium task: write a plan first.
- Large task: create or update `docs/exec-plans/active/`.
- Keep `AGENTS.md` short and operational.
- Put durable facts in `docs/`.
- Put reusable workflows in `.agents/skills/`.
- Finish with verification, not just code changes.
